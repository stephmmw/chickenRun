# Chicken Run: A Game Project

A game using HTML, CSS and JavaScript.

The player controls chicken running from a butcher, while avoiding obstacles, whicle collecting points and other powerups.

## Project Structure

* chicken-run
  * index.html
  * styles/
    * style.css
  * scripts/
    * game.js
    * player.js
    * obstacles.js
    * utils.js
  * images/
